По заданию необходимо мигать светодиодами определенное число раз, 
последовательно мигать каждым светодиодом. Число миганий задаётся с помощью ключей. 
По заданию просят 5 ключей, то есть 5 разрядов, старший разряд отвечает выключение всех светодиодов и завершение работы,
а младшие за число миганий. Для реализации заданного числа миганий строится цикл (1) в который входит ещё один цикл (2).
В цикле (2) считывается значение с ключей (t5) (десятичный формат), которые задаёт число итераций этого цикла. 
В этом же цикле происходит последовательно включение и выключение текущего светодиода.
Цикл повторяется до тех пор, пока счётчик итераций (a1) не станет равным (t5). 
В цикле (2) также предусмотрено включение 5 разряда, если число считываемое с ключей больше или равно 16, то происходит скачок в цикл (4), 
где поочерёдно выключаются все светодиоды. В завершении происходит скачок в (5), 
где снова считывается значение с ключей(в случае динамического переключения ключей) и если выполняется проверка на 5 разряд 
то происходит остановка работы программы. После равенства a1 и t5  в(2) происходит скачок в (3), 
где зажигается текущий светодиод и происходит переход к следующему. Когда зажгутся все светодиоды, с помощью (4) выключатся все светодиоды и всё начнётся заново.
