# подсказки по сборке, запуску и анализу

Запуск машины в режиме дебага:
qemu-system-riscv64 -machine sifive_u -nographic -bios none -S -s -kernel *program_name*

В таком режиме qemu создаст gdb-сервер для отладки и не будет запускать саму программу.
Если нужно просто запустить, нужно убрать флаги -s и -S

Когда машина запущена в режиме дебага, во второй консоли нужно открыть gdb:
riscv64-elf-gdb

В консоли gdb выполнить команды:
target remote :1234
file *program_name*
load

После этого можно работать в дебагере.
--------------------------------------
Для сборки и очистки используются команды make и make clean соответственно.
--------------------------------------
Для получения дизассемблера можно воспользоваться objdump:
riscv64-elf-objdump -D *program_name* > *output_name*
